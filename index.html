Sobre nosotros

Nos enfocamos en Sistemas para Diseño de Presupuestos, extendiendo nuestras opciones en nuevos sistemas para controlar, valuar y dar seguimiento completo al desarrollo de un proyecto. Estamos centrados en desarrollar productos o sistemas para cubrir tres líneas de producción.

Cloud Suite

Ya se encuentra en línea nuestro sistema principal para diseño de Presupuestos por Análisis de Precios Unitarios, funcional desde cualquier dispositivo movil, pc o smart tv. Próximamente se incorporarán en la línea de productos online una serie de sistemas que se integrarán para dar seguimiento, control y manejo de obras y empresa.

Local Sistemas

Puede encontrar una serie de sistemas disponibles para plataforma Windows que le permitirán diseñar y controlar su proyecto u obra. Sistemas disponibles:

- Control de proyectos y obras V1.0
